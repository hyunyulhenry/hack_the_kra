# KRA 경마 기본 규칙
> 예측 모델 개발에 필요한 범위로 정리. 마지막 업데이트: 2026-03-01

---

## 1. 경마장

| 경마장 | 코드 | 시행일 | 연간 경주 수 |
|--------|------|--------|------------|
| 렛츠런파크 서울 | meet=1 | 금·토·일 | ~2,350경주 |
| 렛츠런파크 제주 | meet=2 | 토·일 | ~700경주 |
| 렛츠런파크 부산경남 | meet=3 | 토·일 | ~1,150경주 |

- 1일 경주 수: 보통 11~12경주
- 두수: 8~16두 (보통 12~14두)

---

## 2. 트랙

### 서울
- **주로**: 모래주로 (all-weather)
- **외주로**: 1,800m / **내주로**: 1,600m
- **경주 거리**: 1000 / 1200 / 1300 / 1400 / 1700 / 1800 / 1900 / 2000 / 2300m

### 부산경남
- **주로**: 모래 + 잔디주로 (잔디 별도 운영)
- **경주 거리**: 1000 / 1200 / 1300 / 1400 / 1700 / 1800 / 2000m (잔디: 1400 / 1800 / 2000m)

### 제주
- **주로**: 모래주로 (누운 타원형, 1,600m)
- **경주 거리**: 800 / 900 / 1000 / 1110 / 1200 / 1400 / 1610 / 1700 / 1800m

> ⚠️ 모델 피처: 말마다 거리/트랙/경마장 선호도가 다름 → 과거 성적 기반 적합도 계산 필요

## 3. 등급 체계 (레이팅 기반)

| 등급 | 레이팅 범위 | 부담중량 상한 | 비고 |
|------|-----------|------------|------|
| 1등급 | 81+ | 60kg | |
| 2등급 | 66~80 | 59kg | |
| 3등급 | 51~65 | 58kg | |
| 4등급 | 36~50 | 58kg | |
| 5등급 | 21~35 | 57kg | |
| 6등급 | 20 이하 | 57kg | |
| 오픈 | 제한 없음 | — | 대상특별경주 등 |

- 최저 부담중량: 51kg (전 등급 공통)
- 레이팅은 경주 결과 후 포인트 부여/차감, 비경주 기간에 감소
- 승급 기준: 레이팅 상한 초과 시 → API `raceRating` 필드로 확인 가능

## 4. 부담중량 조정

### 기본 원칙
```
최종 부담중량 = 기본중량 + 부가중량 - 감량
```

### 감량 종류
| 감량 항목 | 조건 | 감량 |
|----------|------|------|
| 암말 감량 | 수·거세마 대비 | -2kg (연중) |
| 2세마 감량 | 타 연령마와 혼합 경주 시 | 시기별 차등 |
| 3세마 감량 | 타 연령마와 혼합 경주 시 | 시기별 차등 (2세보다 소) |
| 기수 견습 감량 | 견습기수(루키) 조건 | 통상 1~3kg |

### 경주 종류별 중량 방식
| 경주 종류 | 중량 방식 |
|----------|---------|
| 핸디캡 경주 | 레이팅 차이 기반 개별 부여 |
| 별정 A/B/C | 성·연령 기초중량 + 부가중량 |
| 마령중량 경주 | 연령·시기별 중량표 적용 |
| 조건 경주 | 등급 내 동일 중량 또는 별정 |

> ⚠️ 모델 피처: 부담중량이 클수록 불리 → 상대적 중량 차이를 피처로 사용

---

## 5. 경주 편성 조건

### 조건 경주 (미승리 → 상위로 올라가는 구조)
- **미승리**: 1승 미만
- **1승 조건**: 1승 이상
- **2승 조건**: 2승 이상
- **오픈**: 승수 제한 없음

### 특수 경주
- **루키(신마) 경주**: 출전횟수 3회 미만인 3세 3월까지의 미승리마
- **대상특별경주**: 최상위 등급, 높은 상금

---

## 6. 기수 규정

- **견습기수 감량**: 루키 자격 기간 중 부담중량 감량 적용 (KRA 내부 기준)
- **기수 변경**: 경주 당일 변경 시 강력한 시그널 (→ Phase 2 API)
- 기수 능력은 최근 1년 성적으로 평가 (→ `기수최근1년성적비교` API)

---

## 7. 착순 및 실격

| 항목 | 내용 |
|------|------|
| 착순 | 1위~출전두수 기록 |
| DQ (실격) | 진로방해, 규정 위반 시 → API `순위비고` 필드 |
| 비완주 | 낙마, 기수 낙승, 경주 포기 |
| 출발 지연 | `출발지연사유` 필드로 기록 |

> ⚠️ 모델 처리: DQ·비완주 경주는 피처 계산 시 제외 또는 별도 처리 필요

---

## 8. 배당률 구조 (예측 모델 핵심)

- 배당률은 **파리뮤추얼 방식** (전체 매출 pool에서 배분)
- 공중 배당률 = 시장 참여자들의 집단 정보 → **벤터 모델의 핵심 변수**
- 경주 전 배당률 시계열 (30분/10분/3분 전) 수집 필요 → race.kra.co.kr 스크래핑

### 환수율
| 승식 | 환수율 |
|------|--------|
| 단승·연승 | 80% |
| 복승·쌍승·복연승·삼복승·삼쌍승 | 73% |

## 9. 주로 상태 (Track Condition)

| 상태 코드 | 의미 |
|---------|------|
| 양호 (Good) | 표준 상태 |
| 다소습 (Yielding) | 약간 습함 |
| 습 (Soft) | 습한 상태 |
| 불량 (Heavy) | 매우 습함, 기록 참고 불가 |

> ⚠️ 모델 피처: 주로 상태별 말의 성적 차이 존재 → 선호 주로 상태 피처 필요
